What does casting mean? 
Casting is the way to convert from a type of variable to another. 
Remember there are 3 most common types of variables: 
strings, integers, and floating. 
In the first secction we can appreciate the variables, 
with the description in a comment next to them. 
In the second secction we call the function casting to convert from one type to another. 
The function cast is very simple, the syntax is the following one:
name_of_varibable = the_type_of_varibale_we_want_to_convert_in(name_of_varibable) 
The last part of the code is designed to print the variables.
